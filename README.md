# Global City Intelligence Platform

A scalable data-driven platform that aggregates open data (World Bank, Eurostat, OpenAQ, etc.) to provide insights about cities and countries.

## Features
- Cost of living data
- Air quality metrics
- Energy data
- City rankings
- Custom scoring system

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Data Sources
- World Bank
- Eurostat
- OpenAQ
